開關機
sudo reboot #重開機
sudo halt   #關機(比sudo poweroff安全)

套件管理
sudo apt-get update             #更新軟體列表
sudo apt-get upgrade            #更新軟體包
sudo apt-get install <軟體名稱>  #安裝套件
sudo apt-get remove <軟體名稱>   #移除套件

安裝Vim編輯器
sudo apt-get install vim
